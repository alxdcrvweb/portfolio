About Me

Hi, my name is Alex! I'm a Frontend Developer with 4 years of experience building full-featured web applications from scratch and optimizing existing codebases.

I specialize in creating fast, responsive, and maintainable user interfaces, focusing on clean architecture and long-term scalability. My core stack includes React, Vue, Next.js, and TypeScript, along with state management tools like MobX, Zustand, and Redux. I use Tailwind CSS, CSS Modules, Sass, and Figma to bring UI designs to life with precision and consistency.

On the backend side, I’ve worked with Node.js, using Express, NestJS, and Next.js API routes to build RESTful APIs and backend services. I’m also familiar with MongoDB, PostgreSQL, and integrating third-party services (like Moralis, Alchemy, etc.) into modern Web3 applications.

I’ve developed and maintained Web3-based platforms, including NFT collections, staking systems, trading dashboards, and social-fi applications. I’m comfortable working with ethers.js, web3.js, wallet integrations, and blockchain data indexing.

In terms of DevOps and infrastructure, I’ve deployed projects on Vercel, VPS, and configured environments using Docker, GitHub Actions, and CI/CD pipelines. I use Git for version control and have worked in collaborative environments, following best practices like feature branching and atomic commits.

I enjoy solving challenging technical problems, improving performance, and refactoring legacy code to align with modern standards.

Currently, I’m looking for opportunities where I can contribute to meaningful projects, grow as an engineer, and work in a team that values code quality, autonomy, and good design.

Projects

Onchain

Onchain Coin is a gamified Web3 platform that rewards users for on-chain activity. It features wallet-based scoring, a tap-to-earn Telegram game, NFT mechanics, and a public leaderboard integrated with smart contracts.

Role: Led the development of the entire frontend infrastructure. Built dashboard interfaces for Onchain Score, smart contract interactions using ethers.js, and created the clicker-based Chain Game interface.

Crypterra

Crypterra is a Web3 card-based game where players mint NFT units and send squads on expeditions to earn in-game currency.

Role: Frontend developer — implemented the NFT minting interface, expedition UI, and wallet integration. Built components for interacting with smart contracts using ethers.js.

Morpheus

Role: Frontend engineer behind smooth UI and seamless blockchain integration. Crafted UX with attention to detail, optimized for performance.

Frensly

Role: Frontend Developer — implemented responsive PWA UI for trading and social features, integrated wallet flows, designed the tiered reward and leaderboard system, and connected real‑time API and smart contract data.

Mine Barons
